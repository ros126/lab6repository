# Lab Git Repository
This program was created as a part of the curriculum of CMPINF0010 for Skills Lab 7. 
This programs displays information from the user.
# Installation
Copy this repository to the machine of choice using:
`git clone https://github.com/ros126/lab6repository.git`
# Usage
Once cloning is complete, you can run the software in any IDE.
The program will ask for your name and age, please input appropriate responses.
The result will then be a statement in the form: "Your name is [name] and you are [age] years old"
